# codex-ui

Interface graphique pour piloter et suivre des sessions Codex synchronisées avec `codex app-server`.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`

Le serveur backend écoute par défaut sur `http://127.0.0.1:3001` et le frontend Vite sur `http://127.0.0.1:5173` en mode développement.
